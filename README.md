# DockerTagWatcher
Python script that sends a notifcation when an Image from Docker Hub gets a new stable update




## Necessary packages :

- Python3
- Python3-requests
- Python3-shcedule
- Python3-smtplib

## Recommended packages :

- Python3-poetry
- Tmux
